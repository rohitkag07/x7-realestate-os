# X7 WhatsAI Assistant - Remotion Video Factory

This Remotion package was built for real-estate content generation. In the WhatsAI pivot, content automation is deferred until the assistant trial and lead conversion loop are proven.

Keep the package available for later vertical assets:

- real-estate project reels
- clinic explainer videos
- coaching admission creatives
- gym/diet challenge creatives
- trial/demo videos for sales

```bash
npm install
npm run dev
npm run render:batch
```

Do not prioritize Remotion work before the WhatsApp assistant trial flow is working end-to-end.
